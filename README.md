# CREARD - Sistema de Gestión de Canchas Deportivas

Plataforma web para la gestión integral de reservas, pagos y control administrativo de canchas deportivas en Cusco, Perú.

## Instalación

```bash
npm install
npx prisma db push
npx prisma db seed
npm run dev
```

## Canchas

- 4 Canchas de Fútbol 5
- 2 Canchas de Vóley
